##Instalação e execução
#### Requisitos:
    Node Js , Expo, Yarn
1. Faça um clone desse repositório;
2. Entre na pasta rodando `cd kiper-challenger`;
3. Rode `yarn` para instalar as dependências;
4. Rode `yarn start` para iniciar o client.
